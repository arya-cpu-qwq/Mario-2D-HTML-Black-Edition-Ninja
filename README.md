# Mario 2D HTML

Proyek sederhana game platformer 2D bergaya Super Mario NES yang dibuat sepenuhnya dengan HTML, CSS, dan JavaScript.

## Fitur

- Tampilan retro ala NES dengan grafis pixel dan palet warna klasik.
- Kontrol pemain untuk berjalan, melompat, dan melakukan double jump.
- Level yang dihasilkan secara dinamis dengan blok bata, kotak pertanyaan, pipa, koin, dan musuh goomba-like.
- Skor, koin, waktu, dan nama pemain ditampilkan pada HUD.
- Layar menu awal bergaya NES dengan input nama pemain.
- Game over dan tombol reset untuk memulai ulang permainan.

## Cara Menjalankan

1. Buka file `index.html` di browser web modern.
2. Masukkan nama pemain di kotak input lalu klik `1 PLAYER GAME`.
3. Mainkan langsung di browser tanpa kebutuhan server atau instalasi tambahan.

## Kontrol

- `Arrow Left` / `A` : Bergerak ke kiri
- `Arrow Right` / `D` : Bergerak ke kanan
- `Space` / `Arrow Up` / `W` : Lompat (dua kali lompat / double jump)

## Catatan

- Game ini ditulis dalam satu file `index.html` untuk kemudahan distribusi.
- Ukuran layar akan menyesuaikan secara otomatis saat browser diubah ukuran.

## Lisensi

Gunakan dan modifikasi sesuai kebutuhan. Untuk penggunaan komersial atau publik, sertakan kredit atau izin sesuai keinginan pembuat.
